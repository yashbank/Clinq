# UI source archive (v0 export)

This folder is a **frozen reference** of the original v0-generated Next app. The **production app** lives at the repository root (`src/app`, `src/components`).

- Do not run `npm install` here for day-to-day work—install at the repo root.
- When refreshing UI from a new v0 export, diff against this tree and merge selectively into `src/`.
